# Medical Shop Automation System

## Overview

Medical Shop Automation System is a web-based application developed to simplify the day-to-day operations of a medical store. It automates inventory management, medicine sales, receipt generation, vendor management, and stock monitoring, reducing manual effort and improving efficiency.

## Features

* Medicine inventory management
* Add, update, and delete medicines
* Vendor management
* Medicine sales with receipt generation
* Expired medicine tracking
* Low-stock (threshold) monitoring
* Revenue analysis
* Search medicines
* User authentication

## Installation

1. Clone the repository

```bash
git clone https://github.com/Arundhathi2425/Medical-Shop-Automation.git
```

2. Navigate to the project folder

```bash
cd Medical-Shop-Automation
```

3. Install the required packages

```bash
pip install -r requirements.txt
```

4. Run the development server

```bash
python manage.py runserver
```

5. Open your browser and visit

```
http://127.0.0.1:8000/
```
